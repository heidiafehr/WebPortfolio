Backtracing Sudoku Solver

Made with: C++

This program computes the solution to a sudoku puzzle using a recursive back-tracing algorithm. The board of the sudoku puzzle is implemented by a 2D array. The recurssive algorithm checks by sector, row and column to see if a number fits. If the number does not fit, it returns false back to the previous call to try a new number until the puzzle is solved.
